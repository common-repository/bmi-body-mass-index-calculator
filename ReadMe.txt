BMI Body Mass Index Calculator Wordpress Plugin

## Creator:
Sexy Londoness has created this plugin for personal use. After many friends have asked for a 
copy, decision was made on the public release.

## Install:
Just install it like any other Wordpress plugin. Upload the plugin's folder to /plugins and
activate it using your wordpress dashboard.
